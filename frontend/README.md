# 🎓 CHOSUN Undergraduate Student's AI Academic Success Champion

A modern React frontend for an AI-powered academic assistant designed to help university students access academic resources, scholarships, projects, and AI-powered academic support.

---

# 📖 Overview

This project provides a user-friendly interface that enables students to:

- 🤖 Interact with an AI Academic Assistant
- 📚 Access academic resources
- 🎓 Browse scholarship opportunities
- 💼 Explore academic projects
- 🏠 Navigate a modern dashboard

The frontend is built using React and Vite and is designed to integrate with a FastAPI backend.

# 🚀 Installation

Clone the repository

```bash

```

Install dependencies

```bash

```

Run the development server

```bash
npm run dev
```
